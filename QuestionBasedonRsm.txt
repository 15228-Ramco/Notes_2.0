2. What areasync/await**, and how do they improve performance?
	Asynchronous programming allows a program to start a long-running task and continue executing other tasks without waiting for the first one to finish. 
	It keeps the application responsive and is especially useful for operations like file I/O and web requests.
		async is a keyword used to mark a method as asynchronous, meaning it can run operations without blocking the calling thread.
		await is a keyword used inside an async method to pause execution until a Task completes, without blocking the thread.
3. What is the difference betweenIEnumerable, IQueryable, ICollection**?
	IEnumerable is used to iterate over in-memory collections and executes LINQ queries in memory.
	IQueryable extends IEnumerable and is used for querying databases or remote data sources, executing queries on the server.
	ICollection also extends IEnumerable but allows modifying the collection with Add, Remove, and Count

Two sum : Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

public int[] TwoSum(int[] nums, int target) {
        
        Dictionary<int,int> numDict=new Dictionary<int,int>();
        int balance;
        for(int i=0;i<nums.Length;i++)
        {
            balance=target-nums[i];
            if(numDict.ContainsKey(balance))
            {
                return [numDict[balance],i];
            }
            else
            {
                numDict[nums[i]]=i;
            }
        }
        return [];
    }
}
